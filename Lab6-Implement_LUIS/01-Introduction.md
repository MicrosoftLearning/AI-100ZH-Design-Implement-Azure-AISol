# 实验 6：实现 LUIS 模型

## 简介

我们将生成一个端到端的方案，它允许你引入自己的图片，使用认知服务查找图像中的对象和人物，以及获取说明和标签。我们稍后将使用 LUIS 生成一个 Bot Framework 机器人，以便轻松且有针对性地进行查询。

> **注：** 在本实验中，我们将仅创建 LUIS 模型，用于在未来的实验中生成更加智能的机器人。
